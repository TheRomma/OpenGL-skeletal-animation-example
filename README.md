# OpenGL-skeletal-animation-example
This is a course project for a university computer graphics course showing skeletal animations working for a custom 3D model format.

## How to build
```
cd stereo-image-depth-estimator
make
./executable
```

## Controls
```
W: Move forward
S: Move backward
A: Move left
D: Move right
Mouse: Look around
M: Lock mouse cursor
ESC: Quit program
```
